A modern, responsive portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## üì¶ Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## üé® Customization

### Personal Information
Update the following files with your information:
- `app/components/Hero.tsx` - Name, tagline, and intro
- `app/components/About.tsx` - Skills, experience, and background
- `app/components/Projects.tsx` - Your projects and portfolio items
- `app/components/Contact.tsx` - Contact information
- `app/components/Footer.tsx` - Social links and contact details

### Styling
- Colors and themes: `tailwind.config.js`
- Global styles: `app/globals.css`
- Component-specific styles: Individual component files

### Content
- Update project images in the `Projects.tsx` component
- Modify skills list in the `About.tsx` component
- Update social media links throughout the components
